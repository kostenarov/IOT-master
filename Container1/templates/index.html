Endpoints:

  • GET / - Get all links
  • POST /data/{car_id} - Post new location to the given car
  • POST /data/cars - Post a new car to the data.json file
  • GET /map/id/{car_id} - Get a map of all the locations in order for a given car
  • GET /map/latest - Get a map of the latest locations of all cars
  • GET /map/furthest - Get the furthest location from a hardcoded one for all cars
